# AndroidChart
Android图表之范围柱状图和雷达图

### 功能特点
基于Android图表库XCL-Charts，根据需求定制自己的范围柱状图和雷达图。主要增加的特点如下：
- 范围柱状图最大最小值相等的时候，画一条横线表示。
- 范围柱状图支持渐变色设置
- 雷达图数据不连续的时候，即中间被0隔开的时候画一条横线表示。
